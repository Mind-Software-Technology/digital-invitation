# Font Laraboyok

Letakkan file font **Laraboyok** di folder ini agar nama mempelai di halaman
cover tampil dengan font yang benar.

Nama file harus salah satu (huruf kecil semua):

- `laraboyok.woff2`  ← paling ringan, disarankan
- `laraboyok.ttf`
- `laraboyok.otf`

Cukup salah satu sudah cukup (boleh lebih dari satu format). Setelah file
ditaruh di sini, jalankan ulang `npm run dev` lalu refresh browser.

Jika file belum ada, font otomatis fallback ke **Playfair Display** —
tampilan tetap rapi, hanya bukan font Laraboyok.

> Catatan: konversi `.ttf`/`.otf` ke `.woff2` bisa lewat
> https://transfonter.org atau https://cloudconvert.com untuk ukuran
> file lebih kecil & loading lebih cepat.
